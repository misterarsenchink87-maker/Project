# Project
Веб-приложение для мониторинга поездов КТЖ с цифровым двойником локомотивов, системой ролей пользователей и авторизацией.

Сайт отображает движение поездов, состояние локомотивов и маршруты на карте.
Доступ к информации зависит от роли пользователя:

Пассажир — видит только маршруты поездов
 Машинист — видит только свой поезд и кабину управления
 Руководитель — имеет полный доступ ко всем данным
 Регистрация и авторизация

3. Email содержит @manager.kz или @admin.kz - тогда вы получаете статус Руководителя
